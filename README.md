# StreamShelf

StreamShelf is a personal media tracker built for my software engineering portfolio. It is inspired by streaming apps, but it solves a more specific problem: keeping track of what I want to watch, what I am watching, what I finished, and what fits my mood.

![StreamShelf dashboard](assets/streamshelf-dashboard.png)

## Why I Built It

I wanted a visual project that felt more creative than a standard dashboard while still showing real application logic. StreamShelf uses mock media data, local storage, filtering, status updates, progress tracking, ratings, and a recommendation-style "Pick Tonight" feature.

## Features

- Browse a mock streaming library
- Filter by genre, platform, and status
- Search by title, platform, genre, or mood
- Track Watchlist, Watching, Finished, and Favorite statuses
- Increment episode progress
- View ratings, runtime, mood, and notes
- Get a "Pick Tonight" recommendation
- Save demo changes in local storage

## Tech Stack

- HTML
- CSS
- JavaScript
- Local storage
- Mock service layer
- Node-based tests
